<h2><b>ğŸ“– About the Project</b></h2>
<p>
This project uses the famous Iris dataset to build a supervised machine learning
model that predicts the species of an Iris flower. The prediction is made using
four numerical features: sepal length, sepal width, petal length, and petal width.
</p>
